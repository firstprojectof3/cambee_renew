# 🐝 Cambee
내 손 안의 대학 생활 정보, 캠비  
**대학생을 위한 생활 정보 제공 + 챗봇 + 캘린더 알림 기능이 포함된 앱입니다.**

## 🛠️ 사용 기술 스택

- **React** (Vite 기반)
- **Tailwind CSS**
- **TanStack Query (React Query)**
- (예정) Firebase / Express / OpenAI API

## 📦 설치 및 실행 방법

```bash
npm install
npm run dev
```

## 📁 폴더 구조 (예정)

/src  
 ┣ /pages            # 페이지 단위 컴포넌트  
 ┣ /components       # 재사용 가능한 UI  
 ┣ /features         # 기능별 모듈 (chat, calendar 등)  
 ┣ /api              # 서버 API 요청 코드  
 ┣ /hooks            # 커스텀 훅  
 ┣ /stores           # 상태 관리  
 ┣ /styles           # 전역 스타일  
 ┗ /assets           # 이미지, 폰트 등 정적 자산
